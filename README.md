# BFHL Hierarchy Analyzer

How to Run
Backend


cd backend
npm install
node server.js
```

Frontend

Open `frontend/index.html` in browser.



API Endpoint

POST `/bfhl`



Features

Validates input format
Handles duplicate edges
Detects cycles
Builds hierarchy trees
Calculates depth
Returns summary
